Python Proxy Scanner
===
#### 简介
fork https://github.com/h01/ProxyScanner

#### 使用
```shell
$ ./ps.py
-v --version    PS version
-f --file       IPS：ip.txt
-i --ips        IPS: 192.168.1.1-192.168.1.100
-p --port       Proxy port (default:8080)
-t --thread     Scan thread (default:10)
-s --save       Save scan result
```
